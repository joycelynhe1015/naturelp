错误记录：
- 报错：ModuleNotFoundError: No module named 'sklearn'
- 位置：analysis.py 第 3 行 `from sklearn.feature_extraction.text import CountVectorizer, TfidfVectorizer`
- 原因：当前 Python 环境未安装 scikit-learn
- 解决：
  1) 执行 `pip install scikit-learn streamlit nltk gensim altair pandas numpy`
  2) 或 `python -m pip install scikit-learn ...`
  3) 如需隔离环境：
     - `python -m venv .venv`
     - `.venv\\Scripts\\activate`
     - `python -m pip install -U pip`
     - `python -m pip install scikit-learn streamlit nltk gensim altair pandas numpy`
  4) 确认：`python -c "import sklearn, streamlit, nltk, gensim, altair; print('OK')"`
  5) 运行：`streamlit run analysis.py`

- 报错：LookupError: Resource 'punkt_tab' not found
- 位置：nltk.sent_tokenize 时尝试加载 punkt_tab
- 原因：NLTK 数据未下载完或路径不对
- 解决：
  - 在 Python 中执行 `import nltk; nltk.download('punkt'); nltk.download('punkt_tab')`
  - 或在代码中已包含自动下载逻辑：
    - `nltk.data.find('tokenizers/punkt')`
    - `nltk.data.find('tokenizers/punkt_tab/english')`
  - 重新运行 `streamlit run analysis.py`
